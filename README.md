# Eco.Green

Eco.Green es un sitio web desarrollado con **HTML5 y CSS3** enfocado en promover el consumo responsable y la sostenibilidad mediante la presentación de productos ecológicos, información corporativa y canales de contacto para los usuarios.

## Descripción del Proyecto

La página web está compuesta por varias secciones interconectadas que permiten al usuario conocer la marca, explorar productos ecológicos, descubrir los principios de la empresa y establecer contacto mediante un formulario.

El diseño utiliza una temática visual inspirada en la naturaleza, con colores verdes y elementos que refuerzan el concepto de sostenibilidad ambiental.

## Funcionalidades

### Página de Inicio
- Presentación de la marca Eco.Green.
- Banner principal con imagen destacada.
- Sección de productos destacados.
- Navegación hacia las demás páginas.

### Catálogo de Productos
- Visualización de productos ecológicos.
- Organización por categorías:
  - Hogar
  - Plantas
  - Cuidado Personal
  - Alimentos
- Barra de búsqueda visual.
- Tarjetas de producto con:
  - Imagen
  - Nombre
  - Precio
  - Estado de disponibilidad

### Página Nosotros
- Historia de la empresa.
- Información sobre la misión y propósito de Eco.Green.
- Presentación de principios corporativos:
  - Consumo responsable
  - Comercio justo
  - Reducción del plástico

### Página de Contacto
- Formulario de contacto.
- Validaciones HTML5:
  - Campos obligatorios.
  - Validación de correo electrónico.
  - Longitud mínima en algunos campos.
- Información de contacto.

## Características del Diseño

- Diseño responsive adaptable a diferentes tamaños de pantalla.
- Navegación intuitiva entre páginas.
- Paleta de colores inspirada en la naturaleza.
- Uso de Flexbox y Grid Layout para la distribución de elementos.
- Efectos hover para mejorar la experiencia del usuario.
